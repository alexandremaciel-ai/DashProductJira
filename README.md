# 🚀 DashProduct - Jira Analytics\n\n## Dashboard de Produtividade Avançado para Equipes de Desenvolvimento\n\n**Transforme dados do Jira em insights poderosos com o sistema de exportação mais completo do mercado.**\n\n---\n\n## ✨ **Principais Funcionalidades**\n\n### 📊 **Dashboard Interativo**\n- Métricas em tempo real de produtividade\n- 4 tipos de gráficos avançados\n- Análise de velocity e cycle time\n- Performance individual e da equipe\n\n### 📋 **Kanban Board Visual**\n- Interface sincronizada com filtros\n- Cards detalhados com todas as informações\n- Status dinâmicos baseados no Jira\n- Modal de detalhes completo\n\n### 🔍 **Sistema de Filtros Robusto**\n- Período (semana, mês, trimestre, customizado)\n- Sprint específico\n- Membro da equipe\n- Tipos de issue (Story, Bug, Task, Epic)\n\n### 📄 **Sistema de Exportação Avançada** ⭐\n- **CSV Profissional:** Dados completos para análises em Excel/Power BI\n- **PDF Executivo:** Relatórios de nível corporativo para apresentações\n- **Relatório Completo:** CSV + PDF com notificações em tempo real\n- **Burndown Data:** Histórico de 30 dias para análises de tendência\n- **Insights da IA:** Análises automatizadas incluídas nos relatórios\n\n### 🤖 **Insights com IA**\n- Análise automatizada de performance\n- Predições e recomendações\n- Identificação de gargalos\n- Sugestões de otimização\n\n### 🌍 **100% Localizado**\n- Interface completamente em português brasileiro\n- Formatos de data e número brasileiros\n- Terminologia técnica localizada\n\n---\n\n## 🚀 **Deploy Rápido**\n\n### **1. Clone e Instale**\n```bash\ngit clone https://github.com/your-repo/DashProductJira.git\ncd DashProductJira\nnpm install\n```\n\n### **2. Configure sua API do Jira**\n1. Acesse [id.atlassian.com](https://id.atlassian.com)\n2. Vá em Account Settings → Security → API tokens\n3. Crie um novo token\n4. Guarde o token com segurança\n\n### **3. Execute**\n```bash\nnpm run dev\n# Para macOS (se porta 5000 estiver ocupada):\nPORT=3000 npm run dev\n```\n\n### **4. Acesse e Configure**\n1. Abra `http://localhost:3000` no navegador\n2. Insira suas credenciais do Jira:\n   - URL: `https://suaempresa.atlassian.net`\n   - Email: seu email do Jira\n   - API Token: o token criado\n3. Selecione um projeto\n4. Comece a usar!\n\n---\n\n## 📊 **Sistema de Exportação em Destaque**\n\n### **📈 CSV Profissional**\n- **15+ seções de dados** estruturados\n- **Compatibilidade total** com Excel/Google Sheets\n- **UTF-8 com BOM** para caracteres especiais\n- **Dados de burndown** dos últimos 30 dias\n- **Estatísticas da equipe** com produtividade individual\n\n### **📄 PDF Executivo**\n- **Design profissional** com cabeçalho e rodapé\n- **Múltiplas páginas** com paginação automática\n- **Cores indicativas** para tendências (verde/vermelho)\n- **Tabelas organizadas** com métricas principais\n- **Pronto para apresentação** em reuniões executivas\n\n### **🎯 Relatório Completo**\n- **Geração simultânea** de CSV + PDF\n- **Notificações visuais** de progresso\n- **Feedback em tempo real** com spinners animados\n- **Tratamento de erros** com sugestões de retry\n- **Downloads automáticos** de ambos os arquivos\n\n---\n\n## 🏗️ **Arquitetura**\n\n```\n📱 Frontend (React + TypeScript)\n    ↕️ TanStack Query\n🔧 Backend (Express + Node.js)\n    ↕️ Proxy Authentication\n🔗 Jira Cloud API\n    ↕️ REST API\n🗄️ PostgreSQL (Produção)\n```\n\n### **Stack Tecnológico**\n- **Frontend:** React 18, TypeScript, Tailwind CSS, Radix UI\n- **Backend:** Node.js, Express, TypeScript\n- **Database:** PostgreSQL com Drizzle ORM\n- **Charts:** Recharts\n- **Build:** Vite\n- **Export:** jsPDF, Custom CSV engine\n\n---\n\n## 📚 **Documentação Completa**\n\n### **Para Usuários**\n- 📖 **[Guia Completo do Usuário](./docs/USER_GUIDE.md)** - Como usar todas as funcionalidades\n- 📊 **[Sistema de
